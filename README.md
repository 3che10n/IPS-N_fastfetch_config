# IPS-N fastfetch config

A config for fastfetch based on IPS-Northstar from the Lancer TTRPG  

Forked from [Lupo-Byte/GMS_fastfetch_config](https://github.com/Lupo-Byte/GMS_fastfetch_config)  

## Requirements

- [Fastfetch](https://github.com/fastfetch-cli/fastfetch)
- Some Symbols require a [Nerd Font](https://www.nerdfonts.com/#home) to be displayed properly

## Install 

```bash
cd ~/.config/fastfetch  
git clone https://github.com/3che10n/IPS-N_fastfetch_config 
```

## Usage

```bash
fastfetch -c IPS-N_fastfetch_config/IPS-N
```
